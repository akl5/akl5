For this project, I decided to study and check out the Apple password manager. I did this truly because of one reason: It's so convenient! For those who are use Apple's devices day-to-day, it's so easy just to use Apple's built in password manager because there's no additional setup or installation neccessary. Apple also generates passkeys and mainly only requires you to use FaceID to access these stored passwords and passkeys. This allows for convenience of the user and is also safer than allowing access to all unlocked passwords after initial device unlock.
